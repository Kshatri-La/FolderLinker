# 🔗 FileLinker (Dual-Site File Manager)
![JavaFX](https://img.shields.io/badge/JavaFX-13-orange.svg) ![JDK](https://img.shields.io/badge/JDK-11+-blue.svg) ![Build](https://img.shields.io/badge/Build-Maven-success.svg)

FileLinker is a high-performance, dual-site file management application built on top of JavaFX. It redefines file browsing with intelligent tree expansion, synchronized visual directory pinning, and lightning-fast bulk export protocols.

> 🇻🇳 Cuộn xuống dưới để xem Hướng dẫn bằng Tiếng Việt.

---

## 🇺🇸 English Guide

### 🌟 Key Features
- **Dual-Pane Interface:** Two completely independent browser areas seamlessly divided, making file transfers hyper-efficient.
- **🔗 Smart Directory Linking:** Click the link `🔗` button next to any folder. The system will intelligently pin strictly matching folders to the absolute top of the container—perfectly aligned across both sites with colorful highlights.
- **Tree Flowchart Engine:** Click standard hierarchical arrows (`▶`, `▼`) to unfurl directories inline without breaking context. Documents have dedicated L-shaped (`↳`) connector cues mimicking flowcharts.
- **Full Drag & Drop:** Drag files or entire folders between panes to transfer. Drop files straight into your exact Windows OS folders. Complete with overwrite warnings.
- **Header Toolbar Engine:** 
  - `[+] New Folder`: Spin up directories into your cursor's exact location.
  - `[❌] Delete`: A verified destruction protocol.
  - `[📤] Export`: Multi-select infinite data via Shift/Ctrl and dispatch them to an external OS volume.

## 🇻🇳 Hướng Dẫn Chi Tiết (Tiếng Việt)

### 🌟 Tính Năng Nổi Bật
- **Giao Diện Kép Song Song (Dual-Pane):** Chia đôi màn hình xài độc lập 2 hệ thống thư mục khác nhau, cho phép dễ dàng luân chuyển dữ liệu như Total Commander.
- **🔗 Cơ Chế Liên Kết (Link):** Click biểu tượng nút xích `🔗` cạnh bất kỳ Thư mục nào. Hệ thống thuật toán sẽ tự gõ tìm đích danh thư mục trùng tên bên bảng còn lại, sau đó **nhấc và ghim chặt** cả 2 folder thẳng hàng nhau trên cùng danh sách. Mã màu rực rỡ để dễ dò.
- **Hiển Thị Cây Sổ Kế Thừa:** Bấm các nút mũi tên (`▶`, `▼`) để xem nhanh ruột thư mục bên trong với lề thụt chính xác. Từng file tài liệu lẻ sẽ có kí hiệu cây cong (`↳`) tách biệt y hệt bản vẽ kỹ thuật.
- **Kéo Thả (Drag & Drop):** Hỗ trợ kéo thả siêu tốc từ cửa sổ này chọi sang cửa sổ kia. Thậm chí có thể bôi đen kéo rớt tung tóe thẳng ra màn hình Windows.
- **Thanh Công Cụ Quyền Lực:** 
  - `[+] New Folder`: Tạo gấp thư mục con.
  - `[❌] Delete`: Xóa (Có hộp thoại báo động nhắc nhở bảo vệ dữ liệu).
  - `[📤] Export`: Giữ phím Shift hoặc Ctrl và rải chuột bôi đen ngẫu nhiên. Bấm nút Mũi tên này để ra lệnh xuất toàn bộ cái đống hỗn độn vừa chọn ra ổ cứng máy tính.
